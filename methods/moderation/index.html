進階

調節分析

調節分析回答「什麼情況下效果比較強或比較弱」。

難度：中高SPSS PROCESSJamoviR / RStudioMplus

適合回答什麼研究問題

某個條件是否改變 X 對 Y 的影響？

  • 學習自我效能是否會改變平台易用性對滿意度的影響？
  • 主管支持是否會緩衝壓力對離職意圖的影響？

需要什麼類型的變項

X：連續或類別自變項W：連續或類別調節變項Y：連續依變項

什麼情況可以用

  • 有明確邊界條件
  • 想檢驗交互作用
  • 調節變項角色清楚

什麼情況不適合用

  • 把中介誤當調節
  • 沒有交互作用卻硬解釋
  • 未中心化或未建立交互項

前提不符合時的替代方法

分組迴歸多群組 SEM中介分析

使用前要檢查什麼

  • 建立 XW 交互作用
  • 連續變項可中心化
  • 畫互動圖或做簡單斜率

常見輸出表格

  • 交互作用係數
  • 簡單斜率
  • Johnson-Neyman 區間
  • 互動圖

常見錯誤

  • 交互作用不顯著卻仍解讀調節成立
  • 沒有畫互動圖
  • 把調節變項誤寫成中介變項

結果怎麼看

  • 看交互作用係數是否顯著
  • 再看簡單斜率
  • 用圖說明效果方向

論文寫作範例句

調節分析結果顯示，學習自我效能顯著調節平台易用性與學習滿意度之間的關係，交互作用項 β = .16, p = .018。簡單斜率分析顯示，在高自我效能組中，平台易用性對滿意度的正向影響較強。

核心公式

調節迴歸模型

Y=b0+b1X+b2W+b3(X×W)Y = b_0 + b_1 X + b_2 W + b_3 (X\times W)Y=b0​+b1​X+b2​W+b3​(X×W)

b3 是交互作用係數，用來判斷 W 是否改變 X 對 Y 的影響。

新手一句話總結

調節分析是問效果在什麼情況下比較強或比較弱。
