Statistics Method Detail

基礎

t 檢定

t 檢定用來回答「兩組平均數有沒有不同」。

難度：中SPSSJamoviJASPR / RStudioPython

適合回答什麼研究問題

兩組人在某個平均分數上是否不同？

  • 男生與女生的學習滿意度是否不同？
  • 實驗組與控制組的後測成績是否有差異？

需要什麼類型的變項

自變項：二分類類別變項依變項：連續分數

什麼情況可以用

  • 依變項通常是連續分數
  • 自變項通常是二分組別
  • 兩組彼此獨立或同一批人前後測

什麼情況不適合用

  • 三組以上比較
  • 依變項是類別資料
  • 樣本太小且分布嚴重偏態時要小心

前提不符合時的替代方法

Welch t-testMann-Whitney UWilcoxon signed-rank testANOVA

使用前要檢查什麼

  • 獨立樣本或成對樣本要先分清楚
  • 檢查變異數同質性
  • 報告平均數、標準差、t、df、p 與 Cohen's d

常見輸出表格

  • 組別平均數與標準差
  • Levene 變異數同質性檢定
  • t 值
  • 自由度 df
  • p 值
  • 效果量 Cohen's d

常見錯誤

  • 把三組以上資料誤用 t 檢定反覆比較
  • 只看 p 值但沒有報告組別平均數
  • 沒有確認兩組是否真的互相獨立

結果怎麼看

  • 先看兩組 M 與 SD
  • Levene 顯著時看 Welch 結果
  • p 值判斷顯著，Cohen's d 判斷差異大小

論文寫作範例句

獨立樣本 t 檢定結果顯示，實驗組的學習成效（M = 4.12, SD = 0.58）顯著高於控制組（M = 3.74, SD = 0.63），t(118) = 3.42, p = .001。

核心公式

獨立樣本 t

t=xˉ1−xˉ2sp1n1+1n2t = \dfrac{\bar{x}_1 - \bar{x}_2}{s_p\sqrt{\frac{1}{n_1}+\frac{1}{n_2}}}t=sp​n1​1​+n2​1​​xˉ1​−xˉ2​​

比較兩組平均數差異相對於組內變異與樣本數的大小。

效果量

d=xˉ1−xˉ2spd = \dfrac{\bar{x}_1 - \bar{x}_2}{s_p}d=sp​xˉ1​−xˉ2​​

Cohen's d 用標準化方式表示兩組平均數差異的實質大小。

新手一句話總結

t 檢定就是兩組平均數比較，但要先確認你真的只有兩組。
