Statistics Method Detail

進階

多元迴歸

多元迴歸用多個因素共同預測一個結果。

難度：中高SPSSJamoviJASPR / RStudioPython

適合回答什麼研究問題

多個因素中，哪些能解釋或預測某個結果？

  • 哪些因素會影響學習滿意度？
  • 控制年齡與性別後，壓力是否仍能預測工作滿意度？

需要什麼類型的變項

自變項：連續或已適當編碼的類別變項依變項：連續變項控制變項：連續或類別變項

什麼情況可以用

  • 依變項是連續分數
  • 有多個自變項或控制變項
  • 想控制其他因素後檢驗主要關係

什麼情況不適合用

  • 樣本數太少但放太多變項
  • 自變項高度重疊
  • 橫斷問卷卻想強烈主張因果

前提不符合時的替代方法

簡單迴歸階層迴歸羅吉斯迴歸SEM

使用前要檢查什麼

  • 檢查 VIF 共線性
  • 檢查殘差與線性
  • 說清楚控制變項理由

常見輸出表格

  • 模型摘要
  • R² 與調整後 R²
  • ANOVA 模型檢定
  • B
  • beta
  • p 值
  • VIF

常見錯誤

  • 自變項過多但樣本數不足
  • 未檢查共線性
  • 把控制變項與主要變項混在一起解釋不清

結果怎麼看

  • Model Summary 看 R²
  • ANOVA 表看整體模型
  • Coefficients 表看每個自變項

論文寫作範例句

多元迴歸分析顯示，在控制年齡與每週使用時間後，知覺有用性仍能顯著預測學習滿意度，β = .38, p < .001，整體模型解釋力為 Adj. R² = .41。

新手一句話總結

多元迴歸是控制其他因素後，看哪些因素仍有預測效果。
