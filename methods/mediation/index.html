進階

中介分析

中介分析回答「影響是透過哪個心理或行為機制發生」。

難度：中高SPSS PROCESSJamoviR / RStudioMplus

適合回答什麼研究問題

X 是否透過 M 影響 Y？

  • 知覺有用性是否透過使用態度影響使用意圖？
  • 職場壓力是否透過情緒耗竭影響工作滿意度？

需要什麼類型的變項

X：連續或類別自變項M：連續中介變項Y：連續依變項

什麼情況可以用

  • 有明確理論機制
  • 想檢驗間接效果
  • X、M、Y 的角色清楚

什麼情況不適合用

  • 沒有理論只想讓模型變複雜
  • 時間順序不合理
  • 樣本數不足

前提不符合時的替代方法

多元迴歸SEM路徑分析

使用前要檢查什麼

  • 確認 X、M、Y 角色
  • 使用 bootstrap 信賴區間
  • 避免過度因果用語

常見輸出表格

  • 直接效果
  • 間接效果
  • 總效果
  • bootstrap 信賴區間
  • 路徑係數

常見錯誤

  • 橫斷資料直接寫成強因果
  • 只用三步驟法不報間接效果信賴區間
  • 沒有說明中介變項的理論角色

結果怎麼看

  • 看間接效果
  • 看 bootstrap CI 是否包含 0
  • 直接效果仍需一起解釋

論文寫作範例句

Bootstrap 中介分析結果顯示，使用態度在知覺有用性與使用意圖之間具有顯著中介效果，間接效果 = .18，95% CI [.10, .29]，信賴區間未包含 0。

核心公式

間接效果

ab(X→M→Y)ab \quad (\text{X}\to\text{M}\to\text{Y})ab(X→M→Y)

a 是 X 對 M 的路徑，b 是 M 對 Y 的路徑，ab 表示間接效果。

新手一句話總結

中介分析是問影響透過什麼機制發生。

難度：中高

同時放入多個自變項，檢驗哪些因素能預測某個連續結果。

適用情境

控制背景變項後，檢驗主要自變項對結果的影響。

需要的變項類型

自變項：連續或已適當編碼的類別變項依變項：連續變項控制變項：連續或類別變項

常用軟體

SPSS、Jamovi、JASP、R / RStudio
