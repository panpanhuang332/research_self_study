Statistics Method Detail

基礎

簡單迴歸

簡單迴歸用一個因素估計某個結果的變化。

難度：中SPSSJamoviJASPR / RStudioPython

適合回答什麼研究問題

一個因素是否能解釋或預測一個結果？

  • 學習動機是否能預測學習成效？
  • 產品知覺價值是否能預測購買意願？

需要什麼類型的變項

自變項：連續變項或二元類別變項依變項：連續變項

什麼情況可以用

  • 一個主要自變項
  • 依變項是連續分數
  • 想看預測方向與解釋力

什麼情況不適合用

  • 依變項是二分結果時
  • 變項關係明顯非線性
  • 想同時控制多個干擾因素時

前提不符合時的替代方法

多元迴歸羅吉斯迴歸相關分析

使用前要檢查什麼

  • 檢查線性
  • 檢查殘差
  • 報告 B、beta、p 與 R²

常見輸出表格

  • 迴歸係數 B
  • 標準化係數 beta
  • t 值
  • p 值
  • R²
  • ANOVA 模型檢定

常見錯誤

  • 沒有檢查線性與殘差
  • 把 R² 解讀為準確率
  • 忽略係數方向與研究假設是否一致

結果怎麼看

  • B 表示自變項增加一單位時結果的改變
  • R² 是解釋變異比例
  • p 值看係數是否顯著

論文寫作範例句

簡單線性迴歸結果顯示，學習動機能顯著預測學習成效，β = .52, t = 8.14, p < .001，模型解釋力為 R² = .27。

核心公式

簡單迴歸模型

y^=b0+b1x\hat{y} = b_0 + b_1 xy^​=b0​+b1​x

用一個自變項 x 預測依變項的估計值。

解釋力 R 平方

R2=1−SSresSStotR^2 = 1 - \dfrac{SS_{res}}{SS_{tot}}R2=1−SStot​SSres​​

表示模型能解釋依變項總變異的比例。
