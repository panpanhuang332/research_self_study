模擬案例

數位學習滿意度研究

平台易用性、知覺有用性與互動品質是否影響學生的數位學習滿意度？

變項設定

自變項
平台易用性、知覺有用性、互動品質
依變項
數位學習滿意度
控制變項
年級、每週使用時間、是否修過線上課程

問卷構面與方法

易用性有用性互動品質學習滿意度
描述統計信度分析EFA相關分析多元迴歸

預期分析流程

  1. 1清理問卷資料
  2. 2描述樣本背景
  3. 3檢查各構面信度
  4. 4探索構面結構
  5. 5建立相關矩陣
  6. 6以多元迴歸檢驗影響因素
模擬案例

社群媒體使用與學習動機研究

社群媒體使用強度是否與學習動機有關，並進一步影響學習投入？

變項設定

自變項
社群媒體使用強度、使用目的
依變項
學習動機、學習投入
控制變項
性別、年級、每日上網時間

問卷構面與方法

社群媒體使用學習動機學習投入
描述統計相關分析多元迴歸中介分析

預期分析流程

  1. 1整理使用時間與量表分數
  2. 2檢查描述統計與極端值
  3. 3分析變項相關
  4. 4以迴歸檢驗預測效果
  5. 5檢驗學習動機的中介角色

  4. 4以多元迴歸檢驗主要假設
