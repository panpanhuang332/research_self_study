Output Guide

統計輸出表判讀圖鑑

SPSS、Jamovi、JASP 的表格常常讓新手不知道該看哪裡。這一頁用文字版表格示範，告訴你哪一欄最重要、哪些數字要寫進論文、常見誤讀在哪裡。

先找任務，再找數字

不同表格回答不同問題。描述統計是在整理資料，t 檢定和 ANOVA 在比較差異，相關與迴歸在看關係或預測，EFA 表格則是在整理量表結構。
先看樣本與變項的基本輪廓。

描述統計表

N平均數 M標準差 SD最小值最大值

哪一欄最重要

平均數與標準差最重要，類別資料則看次數與百分比。

寫進論文

N、M、SD，必要時補最小值與最大值。

p 值判斷

描述統計通常沒有 p 值，它是在整理資料，不是在檢定假設。

效果或解釋力

標準差越大，表示受試者之間差異越大。

常見誤讀

把類別代碼拿來算平均，例如把科系 1、2、3 的平均數拿來解釋。

檢查同一構面的題項是否一致。

信度分析表

Cronbach's alpha題項數修正後項目總相關刪題後 alpha

哪一欄最重要

Cronbach's alpha 與修正後項目總相關。

寫進論文

各構面 alpha、題項數、必要時報項目總相關範圍。

p 值判斷

信度分析通常不看 p 值，而是看 alpha 與題項品質。

效果或解釋力

alpha 太低代表一致性不足，太高也可能表示題目重複。

常見誤讀

只因刪題後 alpha 變高就任意刪題，卻不看題意與理論。

比較兩組平均數是否有差異。

t 檢定表

組別 M組別 SDtdfpCohen's d

哪一欄最重要

先看兩組平均數方向，再看 t、df、p 與效果量。

寫進論文

兩組 M/SD、t(df)、p、效果量。

p 值判斷

p < .05 通常表示兩組差異達顯著。

效果或解釋力

Cohen's d 可補充差異大小，避免只說顯著。

常見誤讀

只報顯著，卻沒有說哪一組比較高。

比較三組以上平均數是否至少有一組不同。

ANOVA 表

SSdfMSFpeta squared

哪一欄最重要

F 與 p 決定整體差異是否顯著，事後比較決定哪幾組不同。

寫進論文

F(df1, df2)、p、事後比較結果、效果量。

p 值判斷

p < .05 代表至少一組不同，不代表每一組都不同。

效果或解釋力

eta squared 可描述組別解釋了多少變異。

常見誤讀

ANOVA 顯著後沒有做 post hoc，卻直接說所有組別都不同。

看兩個連續變項的方向與強度。

相關矩陣

變項名稱rpN

哪一欄最重要

相關係數 r 的正負與大小。

寫進論文

r、p，必要時附相關矩陣表。

p 值判斷

p < .05 表示相關達顯著，但不代表因果。

效果或解釋力

r 的絕對值越大，線性關係越強。

常見誤讀

把「相關」寫成「影響」或「造成」。

看整體模型解釋力。

迴歸 Model Summary

RR²Adjusted R²Std. Error

哪一欄最重要

R² 與 Adjusted R²。

寫進論文

整體模型的 R² 或調整後 R²。

p 值判斷

Model Summary 本身不一定給 p 值，要搭配 ANOVA 表看整體模型是否顯著。

效果或解釋力

R² 是解釋變異比例，不是預測準確率。

常見誤讀

把 R² = .40 寫成模型準確率 40%。

檢查整體迴歸模型是否顯著。

迴歸 ANOVA 表

Regression SSResidual SSdfFp

哪一欄最重要

F 與 p。

寫進論文

F(df1, df2)、p，說明整體模型是否達顯著。

p 值判斷

p < .05 表示整體模型至少有預測力。

效果或解釋力

仍需搭配 Coefficients 表判斷是哪個自變項有作用。

常見誤讀

整體模型顯著就以為每個自變項都顯著。

判斷每個自變項的方向、強度與顯著性。

迴歸 Coefficients 表

BStd. ErrorBetatpVIF

哪一欄最重要

Beta、p、VIF。

寫進論文

主要自變項的 B 或 beta、t、p，必要時報 VIF。

p 值判斷

p < .05 表示控制其他變項後，該自變項仍有顯著預測效果。

效果或解釋力

標準化 beta 可比較不同自變項的相對效果。

常見誤讀

忽略 VIF，沒有檢查共線性。

判斷資料是否適合做 EFA。

KMO and Bartlett's Test

KMOBartlett χ²dfp

哪一欄最重要

KMO 與 Bartlett p 值。

寫進論文

KMO、Bartlett χ²、df、p。

p 值判斷

Bartlett p < .05 表示題項之間有足夠相關，可考慮因素分析。

效果或解釋力

KMO 越高越適合因素分析，通常希望至少達可接受水準。

常見誤讀

KMO 過低仍硬做 EFA。

看題項落在哪些因素上。

EFA Rotated Component Matrix

題項因素 1因素 2因素 3共同性

哪一欄最重要

因素負荷量最高的欄位與跨負荷情形。

寫進論文

保留題項的因素負荷量、因素命名、解釋變異量。

p 值判斷

這張表通常不以 p 值判斷，而看因素負荷量與理論合理性。

效果或解釋力

負荷量越高，代表題項越能代表該因素。

常見誤讀

只看數字自動命名因素，沒有回到題項內容與理論。
