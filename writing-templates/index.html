統計結果怎麼寫成論文段落

統計輸出表不是終點。這裡把常見分析轉成可放進論文結果章的句型、段落與注意事項。

論文句型

描述統計寫法

呈現樣本背景、主要變項平均數與分散程度。

統計表重點

樣本數 N平均數 M標準差 SD次數與百分比

注意事項

  • 類別變項通常報次數與百分比
  • 連續變項通常報平均數與標準差
  • 不要把類別編碼直接當作平均數解釋

論文句型

本研究之［變項］平均數為［M］，標準差為［SD］，顯示［簡短解釋］。

範例段落

本研究共回收有效問卷 286 份。受試者之數位學習滿意度平均數為 3.92，標準差為 0.68，顯示整體滿意度略高於量表中點。

論文句型

信度分析寫法

檢查問卷構面題項是否具有內部一致性。

統計表重點

Cronbach's α題項數項目總相關刪題後 α

注意事項

  • 反向題需先完成反向計分
  • 刪題需搭配理論與題意，不應只看 α
  • 可同時呈現各構面 α

論文句型

［構面］之 Cronbach's α 為［數值］，顯示該構面具有［可接受/良好］之內部一致性。

範例段落

信度分析結果顯示，學習動機構面之 Cronbach's α 為 .87，修正後項目總相關介於 .48 至 .73，顯示該構面具有良好的內部一致性。

論文句型

t 檢定寫法

比較兩組在連續依變項上的平均數差異。

統計表重點

各組 M 與 SDt 值dfp 值效果量

注意事項

  • 先報兩組平均數方向
  • p 值顯著不代表差異一定很大
  • 兩組以上請改用 ANOVA

論文句型

［組別 A］在［依變項］上顯著高於/低於［組別 B］，t(［df］) = ［t］，p = ［p］。

範例段落

獨立樣本 t 檢定結果顯示，實驗組之學習成效（M = 4.10, SD = 0.62）顯著高於控制組（M = 3.72, SD = 0.70），t(128) = 3.25, p = .002。

論文句型

ANOVA 寫法

比較三組以上在某個連續依變項上的差異。

統計表重點

各組 M 與 SDF 值dfp 值事後比較

注意事項

  • 整體檢定顯著後需說明差異在哪些組別
  • 要註明使用的事後比較方法
  • 若變異數不同質需選擇合適替代檢定

論文句型

不同［組別變項］在［依變項］上達顯著差異，F(［df1］, ［df2］) = ［F］，p = ［p］。

範例段落

單因子 ANOVA 結果顯示，不同年級學生在學習動機上達顯著差異，F(3, 232) = 4.91, p = .003。事後比較顯示，四年級學生顯著高於一年級學生。

論文句型

相關分析寫法

檢查兩個連續變項之間是否有線性關係。

統計表重點

相關係數 rp 值方向強度

注意事項

  • 相關不代表因果
  • 建議搭配散佈圖檢查極端值
  • 若資料不符合 Pearson 可考慮 Spearman

論文句型

［變項 X］與［變項 Y］呈現［正/負］相關，r = ［r］，p = ［p］。

範例段落

Pearson 相關分析顯示，學習動機與學習滿意度呈現顯著正相關，r = .51, p < .001，表示學習動機越高者，學習滿意度也傾向越高。

論文句型

迴歸分析寫法

檢驗一個或多個因素是否能預測連續結果。

統計表重點

R²調整後 R²Bbetat 值p 值VIF

注意事項

  • 需說明控制變項
  • 檢查共線性與殘差
  • R² 是解釋變異比例，不是模型準確率

論文句型

［自變項］能顯著預測［依變項］，β = ［beta］，p = ［p］，整體模型解釋力為［R²］。

範例段落

多元迴歸分析顯示，在控制性別與年級後，知覺有用性仍能顯著預測學習滿意度，β = .42, p < .001。整體模型達顯著，Adj. R² = .38。

論文句型

EFA 寫法

探索問卷題項背後的因素結構。

統計表重點

KMOBartlett 檢定因素負荷量解釋變異量轉軸方式

注意事項

  • 需說明萃取方法與轉軸方法
  • 因素命名要回到理論與題項內容
  • 跨負荷題項需謹慎處理

論文句型

KMO = ［數值］，Bartlett 球形檢定達顯著，表示資料適合進行因素分析。

範例段落

探索性因素分析前，KMO 值為 .89，Bartlett 球形檢定達顯著，χ² = 1362.44, p < .001，表示資料適合因素分析。經主軸因子法與 Promax 轉軸後，共萃取三個因素，累積解釋變異量為 64.18%。
