推薦判斷

  • 使用問卷量表時，通常先檢查信度。
  • 兩組平均數差異可先看 t 檢定。

保護性提醒

  • 如果使用問卷量表，不要一拿到資料就直接跑迴歸，應先檢查資料清理、描述統計、信度與效度。
  • 如果只是相關設計，不要把結果寫成因果。
  • 如果樣本數太少，不建議硬跑 SEM、CFA 或複雜中介調節模型。
  • 如果自變項與依變項類型判斷錯誤，統計方法很可能會選錯。

為什麼推薦

同一構面的題項是否穩定地測量同一件事？

論文結果可能怎麼寫

本研究以 Cronbach's α 檢驗量表信度。學習滿意度構面的 α = .89，各題項之修正後項目總相關介於 .54 至 .76，顯示該構面具有良好的內部一致性。

使用前檢查清單

  • 反向題先轉換
  • 檢查項目總相關
  • 刪題需搭配理論

不適合使用的情況

  • 單一題項
  • 題項本來測量不同概念
  • 反向題尚未轉換

t 檢定

看詳細頁

為什麼推薦

兩組人在某個平均分數上是否不同？

論文結果可能怎麼寫

獨立樣本 t 檢定結果顯示，實驗組的學習成效（M = 4.12, SD = 0.58）顯著高於控制組（M = 3.74, SD = 0.63），t(118) = 3.42, p = .001。

使用前檢查清單

  • 獨立樣本或成對樣本要先分清楚
  • 檢查變異數同質性
  • 報告平均數、標準差、t、df、p 與 Cohen's d

不適合使用的情況

  • 三組以上比較
  • 依變項是類別資料
  • 樣本太小且分布嚴重偏態時要小心

替代方法

Welch t-testMann-Whitney UWilcoxon signed-rank testANOVA
